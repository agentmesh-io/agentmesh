## 🔧 Option 1: GitHub Integration (Ready to Implement)

### Why GitHub Integration?
- ✅ No frontend to build/maintain
- ✅ Developers already use GitHub
- ✅ Version control built-in
- ✅ Issue tracking for tasks
- ✅ Pull requests for code review
- ✅ GitHub Actions for automation

### Architecture
```
GitHub Repository
  ├── Issues (Tasks/Features)
  ├── Pull Requests (Generated Code)
  ├── Comments (Agent Communication)
  └── Actions (Trigger AgentMesh)
       ↓
  AgentMesh Backend
       ↓
  Blackboard → Agents → Code Generation
       ↓
  Create PR → Notify Team
```

### Implementation Plan

#### 1. GitHub Webhook Integration
```java
@RestController
@RequestMapping("/api/github")
public class GitHubWebhookController {
    
    @PostMapping("/webhook")
    public ResponseEntity<Void> handleWebhook(
            @RequestHeader("X-GitHub-Event") String event,
            @RequestBody String payload) {
        
        if ("issues".equals(event)) {
            // New issue created → Trigger AgentMesh
            handleIssueCreated(payload);
        } else if ("issue_comment".equals(event)) {
            // Comment added → Trigger specific agent
            handleIssueComment(payload);
        }
        
        return ResponseEntity.ok().build();
    }
    
    private void handleIssueCreated(String payload) {
        // Parse issue
        GitHubIssue issue = parseIssue(payload);
        
        // Post to Blackboard as SRS
        blackboard.post("github", "SRS", issue.getTitle(), issue.getBody());
        
        // Trigger Planner Agent
        triggerWorkflow(issue.getTitle());
        
        // Comment on issue
        githubClient.addComment(issue.getNumber(), 
            "🤖 AgentMesh received your request. Planning in progress...");
    }
}
```

#### 2. GitHub Client
```java
@Service
public class GitHubIntegrationService {
    
    private final RestTemplate restTemplate;
    private final String githubToken;
    private final String repoOwner;
    private final String repoName;
    
    public void createPullRequest(String title, String code, String issueNumber) {
        // 1. Create branch
        String branchName = "agentmesh/feature-" + issueNumber;
        createBranch(branchName);
        
        // 2. Commit code
        commitFile(branchName, "src/main/java/Generated.java", code);
        
        // 3. Create PR
        PullRequest pr = new PullRequest();
        pr.setTitle(title);
        pr.setBody("🤖 Generated by AgentMesh\\n\\nCloses #" + issueNumber);
        pr.setHead(branchName);
        pr.setBase("main");
        
        restTemplate.postForEntity(
            "https://api.github.com/repos/" + repoOwner + "/" + repoName + "/pulls",
            pr,
            PullRequest.class
        );
    }
    
    public void addComment(String issueNumber, String comment) {
        Map<String, String> body = Map.of("body", comment);
        restTemplate.postForEntity(
            "https://api.github.com/repos/" + repoOwner + "/" + repoName + "/issues/" + issueNumber + "/comments",
            body,
            Void.class
        );
    }
    
    public void updateIssueLabels(String issueNumber, List<String> labels) {
        // Add labels like: "agentmesh", "in-progress", "code-generated"
        restTemplate.put(
            "https://api.github.com/repos/" + repoOwner + "/" + repoName + "/issues/" + issueNumber + "/labels",
            Map.of("labels", labels)
        );
    }
}
```

#### 3. GitHub Actions Workflow
```yaml
# .github/workflows/agentmesh.yml
name: AgentMesh Integration

on:
  issues:
    types: [opened, labeled]
  issue_comment:
    types: [created]

jobs:
  trigger-agentmesh:
    runs-on: ubuntu-latest
    if: contains(github.event.issue.labels.*.name, 'agentmesh')
    
    steps:
      - name: Trigger AgentMesh
        run: |
          curl -X POST https://your-agentmesh.com/api/github/webhook \
            -H "X-GitHub-Event: ${{ github.event_name }}" \
            -H "Content-Type: application/json" \
            -d "${{ toJson(github.event) }}"
      
      - name: Comment on Issue
        uses: actions/github-script@v6
        with:
          script: |
            github.rest.issues.createComment({
              issue_number: context.issue.number,
              owner: context.repo.owner,
              repo: context.repo.repo,
              body: '🤖 AgentMesh is processing your request...'
            })
```

### Usage Flow

#### 1. Developer Creates Issue
```markdown
Title: Add User Authentication API

Description:
Create REST API for user authentication with:
- POST /api/auth/login
- POST /api/auth/register
- POST /api/auth/logout
- JWT token-based authentication

Labels: agentmesh, feature
```

#### 2. AgentMesh Processes
```
1. Webhook received
2. Issue parsed → Posted to Blackboard
3. Planner agent creates execution plan
4. Coder agent generates code (with self-correction)
5. Reviewer agent reviews code
6. Test agent generates tests
```

#### 3. AgentMesh Creates PR

```markdown
Closes #123

---
**Review Checklist:**
- [ ] Code quality reviewed
- [ ] Tests passing
- [ ] Security validated
- [ ] Documentation complete
```

#### 4. Team Reviews & Merges
- Review generated code
- Request changes if needed
- Merge PR when approved

### Quick Implementation (React)

#### Dashboard Component
```javascript
// src/App.jsx
import React, { useState, useEffect } from 'react';

function App() {
  const [agents, setAgents] = useState([]);
  const [violations, setViolations] = useState([]);
  const [metrics, setMetrics] = useState({});

  useEffect(() => {
    // Fetch agents health
    fetch('http://localhost:8080/api/mast/health/coder-agent')
      .then(r => r.json())
      .then(setAgents);

    // Fetch violations
    fetch('http://localhost:8080/api/mast/violations/recent')
      .then(r => r.json())
      .then(setViolations);
  }, []);

  return (
    <div className="min-h-screen bg-gray-100">
      <nav className="bg-white shadow">
        <div className="max-w-7xl mx-auto px-4 py-4">
          <h1 className="text-2xl font-bold">AgentMesh Control Panel</h1>
        </div>
      </nav>

      <div className="max-w-7xl mx-auto px-4 py-8">
        {/* Agent Health Cards */}
        <div className="grid grid-cols-3 gap-4 mb-8">
          <AgentCard name="Planner" health={100} status="idle" />
          <AgentCard name="Coder" health={85} status="active" />
          <AgentCard name="Reviewer" health={92} status="idle" />
        </div>

        {/* Recent Violations */}
        <div className="bg-white shadow rounded-lg p-6">
          <h2 className="text-xl font-bold mb-4">Recent Violations</h2>
          <table className="w-full">
            <thead>
              <tr>
                <th>Agent</th>
                <th>Failure Mode</th>
                <th>Severity</th>
                <th>Status</th>
              </tr>
            </thead>
            <tbody>
              {violations.map(v => (
                <tr key={v.id}>
                  <td>{v.agentId}</td>
                  <td>{v.failureModeCode}</td>
                  <td>{v.severity}</td>
                  <td>{v.resolved ? 'Resolved' : 'Open'}</td>
                </tr>
              ))}
            </tbody>
          </table>
        </div>

        {/* Task Creation Form */}
        <div className="mt-8 bg-white shadow rounded-lg p-6">
          <h2 className="text-xl font-bold mb-4">Create Task</h2>
          <TaskForm />
        </div>
      </div>
    </div>
  );
}

function AgentCard({ name, health, status }) {
  const healthColor = health >= 80 ? 'green' : health >= 50 ? 'yellow' : 'red';
  
  return (
    <div className="bg-white shadow rounded-lg p-6">
      <h3 className="font-bold">{name}</h3>
      <div className="mt-2">
        <div className="flex items-center">
          <span className="text-2xl font-bold">{health}</span>
          <span className="text-gray-500 ml-1">/100</span>
        </div>
        <div className={`mt-2 px-2 py-1 rounded text-sm bg-${healthColor}-100 text-${healthColor}-800`}>
          {status}
        </div>
      </div>
    </div>
  );
}

function TaskForm() {
  const [task, setTask] = useState('');

  const handleSubmit = async (e) => {
    e.preventDefault();
    
    await fetch('http://localhost:8080/api/blackboard/entries?agentId=user&entryType=TASK&title=NewTask', {
      method: 'POST',
      headers: { 'Content-Type': 'text/plain' },
      body: task
    });
    
    setTask('');
    alert('Task created!');
  };

  return (
    <form onSubmit={handleSubmit}>
      <textarea
        className="w-full border rounded p-2"
        rows="4"
        value={task}
        onChange={(e) => setTask(e.target.value)}
        placeholder="Describe the task..."
      />
      <button
        type="submit"
        className="mt-2 bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600"
      >
        Create Task
      </button>
    </form>
  );
}

export default App;
```

## 🛠️ Quick Start: GitHub Integration

### 1. Setup Repository
```bash
# Create .github/workflows/agentmesh.yml
mkdir -p .github/workflows
cat > .github/workflows/agentmesh.yml << EOF
name: AgentMesh
on:
  issues:
    types: [opened, labeled]
jobs:
  process:
    runs-on: ubuntu-latest
    steps:
      - name: Trigger AgentMesh
        run: |
          curl -X POST https://your-server.com/api/github/webhook \\
            -H "Content-Type: application/json" \\
            -d '\${{ toJson(github.event) }}'
EOF
```

```yaml
# application.yml
agentmesh:
  github:
    enabled: true
    token: \${GITHUB_TOKEN}
```
